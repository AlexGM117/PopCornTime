# PopCornTime!

## Description

Demo Project for Mobile Coding challenge

## Features
Discover Movies Screen - Paging screen with movie list populated with API call

## Technologies Used

*   Kotlin: 2.1.10
*   Android Gradle Plugin: 8.9.1
*   Jetpack Compose
*   Hilt DI
*   Navigation Compose
*   Ktor HTTP client
*   Coil image loader client
*   Coroutines
*   Paging
